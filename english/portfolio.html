Gear Box Design

We have created this website as our portfolio. The website has been done with custom Html, CSS and javascript. No bootstrap or other stylesheet was used and all the responsiveness was done by creating custom media queries.

THE DESIGN

The Design of the website is clean and focuses on the projects we offer. No unnecessary animations or distracting colours were used, so the customers can solely focus on the design of the projects.


THE ANIMATION

The only animation we have used on this website is an animation of the buttons, so it brings some freshness and interactiveness into the website and creates an eye-pleasing effect for the users.

THE CONTACT FORM

The website contains contact form which was created using bespoke PHP script and works as a form to mail when each submission is passed to our email address where it is stored in the contact folder. It doesn’t use any third party service, but it is solely done in PHP server script. It is also fully responsive for different device screens.


THE PROJECTS

The projects part of the website was design using rows and two columns, which are scaled down to one column when seen on small devices. All code was done using bespoke css and media queries.


Our goals as a front-end designer are to create a clean and sleek website for our customers. A website which suits theirs and their customers' needs. Fully responsive and bespoke.

Thank you for viewing. I hope you enjoyed this project!
