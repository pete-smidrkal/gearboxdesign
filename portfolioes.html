Gear Box Design

Hemos creado este sitio web como nuestro portafolio. El sitio web se ha realizado con HTML personalizado, CSS y javascript. No se usó bootstrap u otra hoja de estilo y toda la capacidad de respuesta se realizó mediante la creación de Media Queris personalizadas.

EL DISEÑO

El diseño del sitio web es limpio y se centra en los proyectos que ofrecemos. No se utilizaron animaciones innecesarias ni colores que distrajeran, lo que permite que los clientes puedan centrarse en el diseño de los proyectos.


LA ANIMACIÓN

La única animación que hemos utilizado en este sitio web es animación de los botones, lo que aporta algo de frescura e interactividad al sitio web y crea un efecto agradable para los usuarios.

EL FORMULARIO DE CONTACTO

El sitio web contiene un formulario de contacto que se creó usando un script PHP personalizado y funciona como un formulario para enviar por correo cuando cada solicitud se pasa a nuestra dirección de correo electrónico, donde se almacena en una carpeta de contactos. No utiliza ningún servicio de terceros, pero se realiza únicamente en el script del servidor PHP. También es totalmente sensible para diferentes pantallas de dispositivos.


LOS PROYECTOS

La parte de los proyectos del sitio web fue diseñada utilizando filas y dos columnas, que se reducen a una columna cuando se ven en dispositivos pequeños. Todo el código se realizó mediante css a medida y Media Queries.


Mi objetivo como diseñador de front-end es crear un sitio web limpio y elegante para nuestros clientes. Un sitio web que se adapte a sus necesidades y las de sus clientes. Totalmente receptivo y a medida.

Gracias por ver. Espero que hayan disfrutado este proyecto!
